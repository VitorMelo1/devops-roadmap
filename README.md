## ⚡ Getting Started

```bash
# Clone the repo
git clone https://github.com/your-username/devops-roadmap.git
cd devops-roadmap

# Go to Step 01 and make scripts executable
cd 01-linux-fundamentals/scripts
chmod +x *.sh

# Run your first script
./01-distros.sh
```

> **Recommended OS:** Ubuntu 20.04+ or Debian 11+. See each step's README for specific prerequisites.
